## 🚀 이미지 ID 추출 방법 (2분 소요)

### 1단계: 각 이미지 우클릭 → "공유" 클릭

### 2단계: "링크가 있는 모든 사용자" 선택 → "링크 복사"

### 3단계: 복사된 링크에서 FILE_ID 추출
```
예시 링크:
https://drive.google.com/file/d/1AbC123XyZ-Example_FILE_ID/view?usp=sharing

FILE_ID = 1AbC123XyZ-Example_FILE_ID
```

### 4단계: 웹사이트 삽입용 URL 생성
```
https://drive.google.com/uc?export=view&id=FILE_ID
```

## ⚡ 빠른 실행 스크립트

FILE_ID를 얻으신 후, 아래 명령어로 자동 업데이트 가능:

```javascript
// 브라우저 콘솔에서 실행 (개발자 도구 F12)
const imageIds = {
  'pet': 'YOUR_FILE_ID_1',
  'space': 'YOUR_FILE_ID_2',
  'kpop': 'YOUR_FILE_ID_3',
  // ... 나머지 ID
};

// 자동으로 배경 이미지 URL 생성
Object.entries(imageIds).forEach(([key, id]) => {
  console.log(`${key}: https://drive.google.com/uc?export=view&id=${id}`);
});
```
